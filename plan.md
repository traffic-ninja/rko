## Документация проекта:

### РЕЗЮМЕ ПРОЕКТА:

# Этап 2: Краткое резюме проекта

---

## 1. Суть продукта

**Агрегатор и сравнительный сервис РКО для российского рынка.** Решает проблему разрозненности информации о тарифах банков и сложности выбора оптимального РКО для конкретной ситуации бизнеса. Целевая аудитория: ИП, самозанятые и ООО — как начинающие предприниматели, так и действующий бизнес, желающий оптимизировать расходы.

---

## 2. Ценность

Единое место для сравнения всех актуальных тарифов РКО с умным подбором под параметры бизнеса (оборот, количество платежей), что экономит время и помогает найти выгодное предложение без изучения десятков сайтов банков.

---

## 3. Ключевой функционал

1. **Каталог банков и тарифов** — структурированная база всех РКО с актуальными условиями
2. **Умный подбор тарифов** — форма с параметрами бизнеса → топ-3 рекомендации с обоснованием
3. **Таблица сравнения** — визуальное сопоставление условий выбранных тарифов
4. **Витрина банков** — спецпредложения и промо от банков-партнеров
5. **Фильтры и сортировка** — поиск по типу бизнеса, региону, условиям обслуживания
6. **Переход на сайт банка** — кнопка для оформления РКО (трекинг для CPA)
7. **SEO-блог** — статьи под коммерческие запросы для органического трафика

---

## 4. Основной user flow

1. **Попадание на сайт** — через поиск Google/Yandex по запросу "сравнение РКО" или "РКО для ИП"
2. **Ввод параметров** — пользователь заполняет форму умного подбора (форма собственности, оборот, кол-во операций)
3. **Просмотр рекомендаций** — система показывает топ-3 подходящих тарифа с пояснениями
4. **Сравнение деталей** — пользователь добавляет варианты в сравнение, смотрит таблицу с условиями
5. **Переход в банк** — клик на кнопку "Открыть РКО" → переход на сайт банка (фиксация конверсии)

2. **Hero-секция** — заголовок с УТП, подзаголовок, форма быстрого подбора (тип бизнеса, оборот, кнопка "Подобрать")

8. **Footer** — ссылки на разделы, соцсети, контакты, копирайт

- **Форма подбора в Hero** — отправляет на страницу "Умный подбор" с заполненными параметрами

- **Карточка акции** — клик → переход на страницу тарифа или витрину
- **Карточка статьи** — клик → переход на статью блога

**Навигация:**
- Откуда приходят: органический поиск, реклама, прямой заход
